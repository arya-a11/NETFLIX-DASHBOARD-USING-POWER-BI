# Netflix User Analytics Dashboard

## Project Overview
The **Netflix User Analytics Dashboard** is a data analytics project designed to analyze user behavior, revenue trends, subscription distribution, and content preferences. The dashboard converts raw user data into interactive visual insights that help understand how users interact with the Netflix platform.

This project demonstrates how data visualization and analytics tools can support business decision-making by identifying trends in user engagement, device usage, payment methods, and popular genres.

---

## Objectives
- Analyze Netflix user behavior using data visualization.
- Understand user distribution across different countries.
- Identify preferred payment methods used by subscribers.
- Evaluate device usage patterns for streaming.
- Analyze user engagement based on age groups.
- Identify popular genres among viewers.
- Provide insights into subscription revenue performance.

---

## Dataset Description
The dataset contains information about Netflix users and their streaming behavior.

### Important Columns

| Column | Description |
|------|-------------|
| User ID | Unique identifier for each user |
| Gender | Gender of the user |
| Age | Age of the user |
| Country | User's country |
| Device Used | Device used for watching Netflix |
| Payment Mode | Payment method used for subscription |
| Subscription Type | Plan selected (Basic, Standard, Premium) |
| Favorite Genre | Most watched genre |
| Revenue | Revenue generated from the user |

The dataset was cleaned and structured before building the dashboard to ensure accurate analysis.

## Key Insights
- Over **19,930 users** are analyzed across **10 countries**.
- **PayPal and UPI** are the most commonly used payment methods.
- **Laptop streaming** is the most frequently used device.
- Users between **30–50 years** show higher content interaction.
- **Drama, Romance, and Documentary** genres are highly popular.
- **Standard subscription plans** generate the highest revenue.
